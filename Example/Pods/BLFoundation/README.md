## Require

- iOS 8.0+ / macOS 10.10+ / tvOS 9.0+ / watchOS 2.0+
- Swift 4.0

## Install

CocoaPod

```

pod 'BLFoundation'

or pod 'BLFoundation/Data'

or pod 'BLFoundation/Date'

or pod 'BLFoundation/Number'

or pod 'BLFoundation/Data'

or pod 'BLFoundation/Tools'

or pod 'BLFoundation/URL'

or pod 'BLFoundation/EventBus'

```
